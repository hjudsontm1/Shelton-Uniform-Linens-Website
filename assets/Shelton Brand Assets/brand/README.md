# Shelton Linen & Uniform Services Brand Assets

This folder contains the approved commercial brand system built around the standalone circled **S** mark. The Shelton Cleaners wordmark and 24/7 kiosk badge are reference material only and are not part of this commercial identity suite.

## Quick Start

| Need | Use |
| --- | --- |
| Website header or wide presentation | `shelton-lockup-horizontal-dark.svg` or `shelton-lockup-horizontal-light.svg` |
| Narrow layout, poster, or square composition | `shelton-lockup-stacked-dark.svg` or `shelton-lockup-stacked-light.svg` |
| App icon, social avatar, seal, or favicon | `shelton-social-avatar-dark.svg`, `shelton-social-avatar-light.svg`, or `shelton-favicon.svg` |
| Vehicle door, route cart, equipment, or signage | `shelton-vehicle-lockup-dark.svg` or `shelton-vehicle-lockup-light.svg` |
| Mark over a photo or flexible CSS color | `shelton-mark.svg` |
| One-color production | One of the `one-color`, `navy`, `cream`, `white`, or `black` files |

## Typography

- Company name: **Cormorant Garamond Medium**
- Descriptor and supporting type: **Inter Medium**

The master circled S is fully path-based. Lockups intentionally retain editable SVG text. Install Cormorant Garamond and Inter before editing, and convert the text to outlines before sending a final production file to a printer, sign fabricator, embroiderer, or vehicle-wrap vendor.

## Usage Rules

- Treat the circled S as the primary logo.
- Keep clear space around the mark equal to at least one-half of the S stroke height.
- Use the standalone mark no smaller than 24 px digitally.
- Use a full horizontal lockup no smaller than 280 px wide digitally.
- Prefer navy, cream, or one-color production versions before using the gold mark.
- Keep the mark upright and preserve its original proportions.
- Use the vehicle lockup only where the descriptor remains legible at the intended viewing distance.

## Do Not

- Do not stretch, skew, rotate, redraw, or add effects to the mark.
- Do not add drop shadows, glows, bevels, or gradients inside the logo.
- Do not place the cream mark on a low-contrast light photograph.
- Do not recolor the logo outside the approved palette.
- Do not use the Shelton Cleaners wordmark as the commercial-services logo.
- Do not use the 24/7 kiosk badge as the company mark.
- Do not send editable-text lockups to production without outlining the type or packaging the fonts.
